📄 Project Abstract

Diabetes is one of the most prevalent chronic diseases worldwide, and early detection plays a crucial role in preventing long-term health complications. This project introduces a complete end-to-end Machine Learning–powered Diabetes Detection System, developed using Python, Scikit-learn, and Streamlit. The system leverages the PIMA Indian Diabetes Dataset to train a logistic regression model capable of predicting the likelihood of diabetes based on patient health attributes such as glucose level, BMI, blood pressure, insulin levels, and more.

The trained model is integrated into a fully interactive Streamlit web application, allowing users to input real-world medical values and instantly receive a prediction accompanied by a probability score. The project follows a modular architecture, including data preprocessing, feature scaling, model training, evaluation, artifact storage, and web deployment. With a clean and extensible folder structure, this project serves as a strong foundation for medical diagnosis applications, academic submissions, or real-world AI-based screening systems.

🧠 What does this project do?

This project builds and deploys a Machine Learning model to detect diabetes based on the PIMA Indian Diabetes Dataset.
It includes:

Automated data preprocessing

Logistic Regression training pipeline

Model evaluation (accuracy + classification report)

Saving trained model, scaler & feature list

A fully interactive Streamlit website

Clean modular folder structure

Ready for deployment / demo presentations
